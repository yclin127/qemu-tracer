Qemu-tracer

QEMU with memory tracer

Download this project as a .zip file Download this project as a tar.gz file

description

A memory tracer done by TCG hack on QEMU. It comes with a configurable cache filter, and python back-end.

instructions

  • ./configure --target-list=x86_64-softmmu --enable-sdl
  • make
  • x86_64-softmmu/qemu-system-x86_64 -hda [image]
    • Ctrl+Alt+q: start/stop tracing
    • Ctrl+Alt+w: print the most recent 256 requests (for debugging)
